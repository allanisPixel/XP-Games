Description
Main Description

O GDD (ou Game Design Document) é um documento contendo todas as informações relevantes do design de um jogo: temática, mecânicas, plataformas, inimigos, levels, entre outros. É de extrema importancia pois registra o escopo do projeto e deve comunicar com clareza os detalhes do projeto atodos os integrantes da equipe.
